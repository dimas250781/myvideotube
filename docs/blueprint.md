# **App Name**: MyVideoTube

## Core Features:

- Video Playback: Play videos with standard controls (play, pause, volume, fullscreen).
- Video Search: Allow users to search for videos using keywords.
- Channel Subscriptions: Enable users to subscribe to channels.
- Personalized Recommendations: AI-powered recommendations to suggest videos based on viewing history; uses viewing patterns as a tool to decide how heavily to weight certain variables.
- Responsive Design: Ensure the web app is fully responsive across desktop, mobile, and tablet devices.

## Style Guidelines:

- Primary color: Dark grey (#282828) to provide a sleek and modern dark mode experience, similar to YouTube Premium.
- Background color: Almost-black (#121212), close in hue to the primary color, for a seamless, immersive viewing environment.
- Accent color: Bright red (#FF4B4B), an analogous hue to orange but with more vivid brightness and saturation, used to highlight interactive elements like the play button and subscription options.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its modern, neutral look.
- Use icons that are consistent with the YouTube style, focusing on clarity and recognizability.
- Implement a grid-based layout to ensure content is well-organized and easily accessible across all devices.
- Incorporate subtle transitions and animations to enhance user experience when navigating between pages or interacting with elements.
